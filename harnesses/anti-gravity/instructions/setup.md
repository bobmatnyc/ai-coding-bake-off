# Anti-Gravity Harness Setup

## Prerequisites

- VS Code with Anti-Gravity extension
- Python 3.12+

## Configuration

1. Open the project root in VS Code with Anti-Gravity
2. No special instruction file needed --- use the prompt directly

## Running a Challenge

1. Open `prompts/level-{N}-prompt.md`
2. Instruct Anti-Gravity: "Read this prompt and solve the challenge. Put your solution in harnesses/anti-gravity/output/level-{N}/"
3. Let the agent work

## Output

- Solutions in `output/level-{N}/`
- Metadata in `output/level-{N}/metadata.json`

## Reproducing

1. Clone the repository
2. Install VS Code with Anti-Gravity extension
3. Open the project and provide the prompt
4. Results appear in `harnesses/anti-gravity/output/`
